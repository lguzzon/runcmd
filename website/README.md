# RunCmd Website (Astro + Tailwind)

Static marketing site for the RunCmd project, built with Astro and Tailwind CSS. Bun is the package manager/runtime.

## Prerequisites
- [Bun](https://bun.sh/) >= 1.3.5

## Install
```bash
bun install
```

## Commands
```bash
bun run dev     # start dev server
bun run lint    # eslint
bun run check   # astro type/diagnostics
bun run build   # outputs to ../public for GitHub Pages
```

## Notes
- Type declarations live in `src/env.d.ts` and are generated by Astro.
- Build artifacts are emitted to the repository-level `public/` directory for GitHub Pages.
